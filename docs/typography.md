# Типографическая система

Основную информацию по типографической информации можно найти [здесь](https://wikidev.chulakov.org/frontend/typography/typography.md) <br>
Сгенерировать код по типографической таблице можно на [этом сайте](https://chtcalculator.chulakov.dev/typo/)

## Что необходимо получить от дизайна для успешной реализации системы?

✅ Таблица с переменными (обязательно!)

✅ Значения для всех брейкпойнтов

✅ Line-height для каждого элемента

✅ Согласованность в использовании

## Как это работает ?

Со стороны дизайна вы получите значения в px. Для начала нужно перевести эти значения в `rem`, разделив полученные значения на 16 (дефолтное значение `rem` при размере шрифта в браузере "Средний") и обновить переменные в `:root`.:
